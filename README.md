# Сравниваем вакансии программистов
Программа сравнивает вакансии для программистов на двух сайтах о поиске работы (HH.ru и SuperJob)

## Как установить
1. Скачайте репозиторий: https://github.com/VladUshakov97/API_lesson5
2. Установите зависимости:
```bash
pip install -r requirements.txt
```
3. Запустите программу
```bash
python main.py
```
## Цель проекта
Код написан в образовательных целях на онлайн-курсе для веб-разработчиков dvmn.org.
